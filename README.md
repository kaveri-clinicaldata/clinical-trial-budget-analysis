# Clinical Trial Budget Analysis (Python)

## Project Overview
This project demonstrates basic data analysis on clinical trial budget and accounts receivable (AR) data.

The goal is to simulate real-world financial tracking used in Clinical Trial Finance.

## Skills Demonstrated
- Data Cleaning using Python
- Excel Data Analysis
- Financial Tracking (AR & Budget)
- Healthcare Data Handling
- Basic Automation mindset

## Dataset Description
Sample dataset includes:
- Study ID
- Sponsor Name
- Invoice Amount
- Paid Amount
- Outstanding AR
- Visit Costs
- Procedure Costs

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Why this project?
As a Clinical Trial Finance Associate, I work with budgets, invoices, and AR reconciliation.  
This project shows how Python can be used to automate and analyze financial workflows in clinical research.
